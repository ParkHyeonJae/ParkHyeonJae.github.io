## 저급언어

- **컴퓨터**가 이해하기 쉽게 작성된 프로그래밍 언어
- 예)
  - 기계어
  - 어셈블리어
- 특징
  - 속도가 매우 빨름
  - 사람이 배우기 까다로움



## 고급 언어

- 컴퓨터보다는 사람이 알기 쉽도록 작성된 프로그래밍 언어
- 예)
  - C++
  - Java
  - C
- 특징
  - 컴파일러나 인터프리터에 의해 기계가 이해 할 수 있는 언어로 번역되어 실행 된다.
  - 고급언어로 작성된 코드 -> 컴파일러 & 인터프리터 -> 기계어 (일부 생략함)
  - 가독성이 높다
